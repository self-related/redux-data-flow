# Redux Data Flow
Демонстрация работы React, Redux Toolkit, React Router с базовыми настройками SEO (title и description для отдельной страницы) и адаптивным дизайном.

## Особенности:
- Структура проекта согласно [Feature-Sliced Design](https://feature-sliced.design) 
- Адаптивный дизайн без пикселей - только флексы и rem
- Базовый логин с редиректом
- Редактирование и удаление постов
- Динамические title и description в зависимости от содержимого

## Стек технологий:
- React
- React Router
- Redux Toolkit
- Typesctipt
- CSS-модули
